# peanut-industry

## vision

Terminal-based idle factory game. You run a peanut butter operation, buy raw peanuts, roast them, sell jars, manage workers and markets.

Automate your factory by writing `.ias` (industry-automation-script) scripts in your editor of choice.
Files hot-reload instantly. The game runs in the background while you're away.

## stack

C + Ncurses + No AI
